Version 3

Released on 10/10/2012

Revision number ( SVN) : 582

Some new features are:

1. In Administration:

a. Move groups and roles management from Google App to inside Health Monitoring System, before this is depended on Google Apps

b. Allows to synchronize user accounts from Google app to monitoring system

c. Proxy link: this allows Health Monitoring Tool retrieve data from remote system via the Proxy link in case of unsigned certificate failure ( SSL )

2. In Dashboard:

         a.  User only sees monitoring system on dashboard if the user belongs to the group which is setup for the monitoring system, before user can see all minoring systems on the dashboard    regardless of these systems in which groups
       
         b. "Real time" data showing: in system information page, some graphs now shows a "real time" data, and movement of graphs

3. Email notification: improve the alignment, and wording

4. Added reversion page: this allows users to see the evolution of the monitor tool
